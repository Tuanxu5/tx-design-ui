## Setup

<Tabs>
  <TabItem value="npm" label="Npm" default>
    <CodeBlock language="jsx">npm install --save tx-design-ui</CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    <CodeBlock language="jsx">yarn add tx-design-ui</CodeBlock>
  </TabItem>
</Tabs>

## Usage

```jsx npm
import React, { Component } from "react";
import { TxMyComponent } from "tx-design-ui";

class App extends Component {
  render() {
    return <TxMyComponent {...props} />;
  }
}
```
